# 📊 タスク・営業目標統合管理ダッシュボード「Momentum」

## 💻 概要
本アプリケーションは、日々のタスク管理（カレンダー・優先度・顧客へのコミットメント・帰宅前ルーティン）と、人材紹介営業における売上目標・進捗率（Q実績および返戻金の計算を含む）を一画面でリアルタイムに可視化・一元管理できる、HTML/CSS/JavaScript製のシングルページアプリケーション（SPA）です。

## 🎯 開発の背景と解決した課題
一般的なタスク管理ツールでは、「時系列の予定」「優先度別のタスク」「顧客への重要な約束」が分散しやすく、また営業職にとって最重要である「売上数値の進捗」と連動していない点に不便さを感じていました。加えて、業務バースト時のリスクヘッジとして「帰宅前に必ず行うべき定常業務」のチェック漏れを防ぐ仕組みも必要でした。
これらの課題を解決し、ビジネスの進捗状況と個人のタスクを完全同期させて生産性を最大化することを目的に、本ダッシュボードを自作しました。

## ✨ 技術的なアプローチ・実装ポイント
- **localStorageを活用したデータの永続化とポータビリティ：** データベースサーバーを不要とし、ブラウザの`localStorage`を用いたサーバーレスでのデータ永続化を実現。また、環境移行やバックアップを想定し、JSON形式でのデータ一括エクスポート・インポート機能を独自に実装しています。
- **非同期での定時監視リフレッシュ処理：** `setInterval`を活用し、バックグラウンドで常に現在時刻とタスクの期限を照合。ユーザーの入力を妨げることなく、期限超過（Overdue）したタスクを検知し、CSSアニメーションを伴うアラート表示へ動的に切り替える状態管理を行っています。
- **誤操作防止のためのUndo（元に戻す）機能：** タスク完了時の誤操作によるユーザーのストレスを軽減するため、画面下部にポップアップする `Undo` トースト通知機能を実装。10秒間のタイマー制御と直前のアクションデータのキャッシュ管理ロジックを構築しました。
- **データドリブンな進捗計算ロジック：** 人材紹介の選考ステータス（未提出〜決定）に応じた柔軟なカラーコーディング。ステータスが「決定」に変更された際、即座にタイムスタンプ（`settledAt`）を付与し、Q目標に対する実績値、残り金額、達成率をリアルタイムで自動計算して画面のインジケーターに反映します。
